homebrew-dreal
=============

Homebrew tap for [dReal4][dreal4]

[dreal]: https://github.com/dreal/dreal4

How to Install
--------------

```bash
brew tap dreal/dreal
brew install dreal
```

Run ``brew upgrade dreal`` to get the latest version of dReal after installation.
