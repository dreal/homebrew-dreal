homebrew-dreal
=============

Homebrew tap for [dReal][dreal]

[dreal]: https://dreal.github.io

How to Install
--------------

```bash
brew dreal/dreal/dreal
```

Run ``brew upgrade`` to get the latest version of dReal after installation.
